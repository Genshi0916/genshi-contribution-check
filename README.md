# genshi-contribution-check
 
自分がその日コミットしていなかったら#contributter_reportを利用して煽り散らす自分用のTwitter Botです

# Requirement
 
* python 3.8.5
* pip 20.2.3
 
# Installation
 
```bash
$ pip3 install tweepy
```
# Important points
key.py must be run with your API key and an access token
 
# Usage
```bash
$ python3 twitter_bot.py
```
